## 典型项目结构（模板）

对于新项目，使用本工具箱后的产物结构如下：

```text
project-root/
├── 01-需求与目标-PRD.md
├── 02-视觉风格与首要页面效果图.md
├── 03-UI规范文档.md
├── 04-组件实现说明.md
├── 05-截图对比报告.md
├── 06-交付清单.md
├── mockups/
│   ├── 01-primary-首页工作台.png
│   └── 02-primary-用户管理.png
└── src/
    ├── api/
    ├── components/
    ├── layouts/
    ├── modules/
    └── ...
```

## 阶段产物说明

| 阶段 | 产物 | 用途 |
|------|------|------|
| 1 需求与目标 | `01-需求与目标-PRD.md` | 产品需求文档 |
| 2 视觉探索 | `02-视觉风格与首要页面效果图.md` + `mockups/*.png` | 视觉方向确认 |
| 3 UI Spec | `03-UI规范文档.md` | 实施规格 |
| 4 组件化实现 | `04-组件实现说明.md` + 可运行代码 | 前端实现 |
| 5 截图对比修正 | `05-截图对比报告.md` | 质量验证 |
| 6 交付与沉淀 | `06-交付清单.md` | 项目收官 |
